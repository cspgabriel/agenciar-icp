# Keywords Negativas — AgenciAR

## 🚫 Lista Consolidada de Negativação
Categorizada por intenção e contexto para proteger o budget do cliente.

| Categoria | Keyword Negativa | Justificativa |
|-----------|------------------|---------------|
| **Gratuidade** | grátis, gratuito, free, graça, sem custo, sem investimento | O cliente busca SQLs (contatos dispostos a pagar por serviço premium). |
| **Educacional** | curso, aula, tutorial, how to, como fazer, dicas, aprender, guia, diy, mentoria, treinamento | Focar em quem quer contratar execução, não em quem quer aprender a fazer. |
| **Emprego** | vaga, emprego, estágio, currículo, trabalhe conosco, recrutamento, freelancer, job | Evitar candidatos a emprego gastando cliques de prospecção. |
| **Específicos Nicho** | wix, canva, shopify, nuvemshop, herospark, hotmart, braip | Pessoas buscando plataformas específicas ou modelos de infoproduto puro. |
| **Modelos Diferentes** | dropshipping, sorteio, rifa, pirâmide, marketing multinível, mmn, afiliados | Modelos de negócio que não batem com o perfil de PME escalável da AgenciAR. |
| **Informacional** | o que é, significado, conceitos, história, wikipedia, blog | Topo de funil muito distante da conversão imediata. |
| **Tecnologia/TI** | python, script, código, api google (desenvolvimento), github | Desenvolvedores buscando documentação técnica, não gestão de mídia. |
| **Concorrência Irrelevante** | agência v4, agência v5, agência rocket, agência lobo | Leads buscando nomes específicos de concorrentes gigantes com marcas proprietárias. |
| **Qualificadores Baixos** | barato, promoção, desconto, cupom, oferta, liquidação | Posicionamento premium (R$ 997+ plano básico) não combina com busca por preço. |
| **Localidade (Fora)** | são paulo, belo horizonte, curitiba, porto alegre | Foco inicial é RJ e Brasil (geral), mas negativamos cidades específicas se o foco for local RJ. (Ajustar conforme campanha). |
